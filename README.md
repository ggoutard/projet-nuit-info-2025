# projet-nuit-info-2025

Ce projet propose d'explorer un système d'exploitation.

Une fois la session ouverte, il est possible de lancer des activités en cliquant sur les icônes sur la gauche.






# Easter eggs à découvrir (SPOILERS) :

Le combo de touche CTRL + ALT + S permet d'accéder à un jeu Snake.

Le Konami Code (Flèche du haut, flèche du haut, flèche du bas, flèche du bas, flèche de gauche, flèche de droite, flèche de gauche, flèche de droite, b, a) utilisé durant le formulaire de contact permet d'activer le mode "disco"
